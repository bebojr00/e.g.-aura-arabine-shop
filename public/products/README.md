# Product images

Official product photography synced from **ASSAF** (`assaf.ae`) and **IBRAQ** (`usaibrahimalqurashi.com`) Shopify stores, optimized as JPEG for the web.

## Re-sync images

```bash
npm run images:sync
```

Requires Node.js and `sharp` (dev dependency).
